### 「雞本創客」第一堂課：雞舍重建計畫 (90分鐘)

**課程目標：**

1.  建立團隊共識，理解雞舍重建的目標與重要性。
2.  透過分工，讓所有學生都能參與專案。
3.  學習觀察使用者(雞)需求，並將其轉化為設計想法。

| 時間 (分鐘) | 活動流程 | 地點 | 帶領者 | 活動內容與提醒 |
| :--- | :--- | :--- | :--- | :--- |
| **15** | **點名 & 課前引導** | 教室 | 兩位老師 | 1. 點名、確認學生狀況。\<br\>2. 播放颱風破壞雞舍的照片或影片，引導學生回憶當時狀況。\<br\>3. **宣布今日超級任務：**「我們要分工合作，啟動雞舍重建計畫！」\<br\>4. **說明分組：** 介紹「建築隊」與「設計監工隊」的任務，強調兩隊同等重要。 |
| **5** | **移動 & 任務說明** | 前往雞舍 | 兩位老師 | 1. 帶領學生移動到雞舍現場。\<br\>2. **現場震撼教育：** 讓學生親眼看看損壞的雞舍，感受重建的必要性。\<br\>3. 發下工具與學習單，再次簡述兩隊任務後，正式分組。 |
| **25** | **分組任務 (一)** | 雞舍周圍 | **您：** 建築隊\<br\>**協助老師：** 設計監工隊 | **建築隊 (4-6人)：**\<br\>  - 安全第一！講解工具(手套/鋸子/鐵鎚)使用安全。\<br\>  - 任務：從清理、拆解可用的舊木料開始。\<br\>**設計監工隊 (6-8人)：**\<br\>  - 任務：使用「使用者需求訪談學習單」進行觀察與紀錄。 |
| **10** | **期中匯報 & 設計交流** | 集合點 | 兩位老師 | 1. 請「設計隊」的同學分享他們的初步觀察與發現（雞喜歡在哪？舊雞舍哪裡最糟？）。\<br\>2. 請「建築隊」的同學分享施工進度與遇到的困難（例如：這塊木頭濕了不能用）。\<br\>3. **腦力激盪：** 根據雙方回饋，大家一起討論新設計有沒有要修改的地方。 |
| **20** | **分組任務 (二)** | 雞舍周圍 | **您：** 建築隊\<br\>**協助老師：** 設計監工隊 | **建築隊：**\<br\>  - 根據剛剛的討論，繼續進行測量、標記或初步組合。\<br\>**設計監工隊：**\<br\>  - 將剛剛討論的結果，畫成更完整的「新雞舍設計草圖」。\<br\>  - 開始思考材料清單：我們需要多少木板？幾根釘子？ |
| **15** | **總結 & 環境收拾** | 集合點 | 兩位老師 | 1. **成果總結：** 各隊派代表展示今天的成果（一堆整理好的木料、一張充滿想法的設計圖）。\<br\>2. **工具清點與環境清潔：** 這是創客精神最重要的一環！帶領所有學生一起完成。\<br\>3. **預告下次：** 「下次，建築隊就要根據設計圖開始動工囉！」 |

### 雞舍重建專案：使用者需求訪談學習單

**團隊名稱：** 雞舍設計/監工隊　　**調查員姓名：** \_\_\_\_\_\_\_\_\_\_\_\_\_　　**日期：** \_\_\_年\_\_\_月\_\_\_日

**我們的任務：** 成為雞的代言人！設計一個更安全、更舒服、更棒的家！

#### **第一部分：我們的「使用者(雞)」是誰？**

請仔細觀察雞，並把你的發現畫下來或寫下來。

| 觀察項目 | 你的發現 (畫下來或寫下來) |
| :--- | :--- |
| **牠們喜歡待在哪裡？**\<br\>(陰涼處？高處？草地上？) | |
| **牠們會做什麼事？**\<br\>(散步、吃東西、抓癢、休息？) | |
| **牠們的身高、體長大約多少？**\<br\>(可以用你的手或尺比比看) | |
| **牠們吃飯/喝水的樣子？**\<br\>(會不會搶？頭要伸多低？) | |

#### **第二部分：舊雞舍為什麼會壞掉？**

請像偵探一樣，找出舊雞舍的弱點！

| 觀察項目 | 你的發現 (畫下來或寫下來) |
| :--- | :--- |
| **哪一個地方壞得最嚴重？**\<br\>(屋頂？牆壁？柱子？) | |
| **你覺得它為什麼會從這裡壞掉？**\<br\>(被風吹？被雨淋濕？木頭太細？) | |
| **舊雞舍有沒有不好用的地方？**\<br\>(很難撿雞蛋？很難清理雞大便？) | |

#### **第三部分：新雞舍的超級點子！**

根據上面的觀察，我們來設計一個更厲害的新家！

| 設計項目 | 我們的「雞」妙點子 (畫下來或寫下來) |
| :--- | :--- |
| **新雞舍的「門」要開在哪？多大？** | |
| **吃飯和喝水的地方要怎麼設計？** | |
| **睡覺的「棲木」要裝在哪裡？多高？** | |
| **下蛋的地方需要什麼特別設計嗎？**\<br\>(是不是要暗一點、舒服一點？) | |
| **【腦力激盪】**\<br\>我們能不能加一個超酷的功能？\<br\>(例如：自動滑水道讓雞蛋滾出來？) | |

**給老師的提示：**
可以多印幾張，讓學生兩人一組共同完成一份，鼓勵他們互相討論。這份學習單不僅是「工作」，更是未來課程成果發表會上，非常有價值的「專案歷程記錄」！
